Installation IntraSell classic 2.08

(This file is written in German, English and Bulgarian language)
===============================================================================



Deutsch - Installation 
----------------------
  1. Archivdatei entpacken in einem beliebigen Verzeichnis
  2. Datei IntraSell_*.mde starten (Initialpasswort: 123) 

Voraussetzung: MS Access 2000 (DE), MS Word 2000

Programmstörungen: die neueste Windows und Office Service Packs installieren!

http://windowsupdate.microsoft.com
http://office.microsoft.com/germany/productupdates/

für Hilfe Email an: office@vegsys.com
===============================================================================



English - Instalation 
----------------------
  1. Unzip the downloaded file
  2. Click on file IntraSell_2.07.mde to start (123 is current Password) 
  3. Change the language - default is DEU

You need: MS Access 2000, MS Word 2000

On any error: please download and install the latest Windows and Office Service Packs

http://windowsupdate.microsoft.com
http://office.microsoft.com/germany/productupdates/

for Help Email to: office@vegsys.com
===============================================================================



Bulgarski - Instalacionen Prozes
--------------------------------
  1. Razarhiviraite failovete na harddiska
  2. Startirajte faila IntraSell_2.07.mde (123 e parvonachalnata parola) 
  3. smenete ezika, ako e neobhodimo - parvonachalno e DEU

Na Was wi e nuzno: MS Access 2000, MS Word 2000

Ako ima problemi: instalirajte naj-nowite Windows i Office Service Packs:

http://windowsupdate.microsoft.com
http://office.microsoft.com/germany/productupdates/

ako imate nuzda ot pomost pratete Email na: office@vegsys.com
