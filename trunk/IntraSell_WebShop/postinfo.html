Web Publishing-Information

Die HTML-Kommentare auf dieser Seite enthalten Konfigurationsdaten, mit deren Hilfe Benutzer mit dem Microsoft Web Publishing-Assistenten oder Programmen, die den Microsoft Web Publishing-Assistenten verwenden, wie z.B. FrontPad, Seiten in Ihrem Web bearbeiten und dabei denselben Benutzernamen und dasselbe Kennwort wie bei der Dokumenterstellung mit FrontPage Explorer und FrontPage Editor verwenden können. Der Microsoft Internet SDK enthält weitere Informationen über die APIs des Web Publishing-Assistenten.
