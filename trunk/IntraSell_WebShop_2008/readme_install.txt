  Installation Instructions
  
  for Web Application "IntraSell  WebShop"



1. Download and install MySQL Driver 32 bit, event if you have windows 64 Bit. 

2. Create WebApp with Home Folder IntraSell_WebShop_2008\WebShop_2008\

3. Set the application Pool to .NET4 and 32 Bit Support

4. Register/install the components from IntraSell_WebShop_2008\WebShop_2008\components folder. 

5. Switch off in web.conf the customer errors to off to get debug error messages in browser.
